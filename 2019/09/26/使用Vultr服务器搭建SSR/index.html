Vultr搭建过程

  这个教程基本和网上的一模一样把，就是我自己记录一下然后之后要用的时候直接来这儿翻，方便一点吧，毕竟感觉这个还是挺容易挂的(可能最近分头比较紧吧，ohhhhh)。

1.创建服务器

  就是去Vultr官网创建一个服务器嘛，之前是有那种冲10$送50$的活动的，但是好像那50$只能在第一个月使用，第二个月就清空了，但是现在好像没有了啊，但是可以时刻的关注一下活动，说不定有什么白嫖的机会也说不定呢…活动就是在官网里面。确实是挺贵的如果没有优惠的话，一个月5$顶不住的啊，比我之前的那个搬瓦工的还难顶啊。

  选择Alipay就好了。

  然后点击加号即可创建服务器，选择服务器的地点，然后选择CentOS7.0即可，之后选择价格，好像不能选2.5$的服务，只支持IPV6，就选自己喜欢的就好了。其他都是默认就好了。创建完服务器之后等待1min即可查看相应信息。

  这里的IP和密码为登录服务器时候的IP和密码，均可直接复制使用。

  因为有可能给你的服务器IP就是挂了的，所以就要重新选取然后直到获取到可以使用的IP即可，但是之前开启了的服务器不可以随便的关闭，要distory才能够不再消耗钱包里面的钱，如果只是停用关机而不销毁服务器的话，还是会按照既定价格收费。

SSR搭建

  直接使用xshell进入服务器即可，IP和密码为上图中服务器IP和密码。

注意：连接服务器不是用Vultr网站登录密码,而是服务器密码,假如连不上服务器,可能是IP被墙,或者是TCP阻断了,建议重新创建服务器,并且删除原有的.

  然后进入之后，输入搭建的代码即可。

	wget --no-check-certificate https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocksR.sh

	chmod +x shadowsocksR.sh

	./shadowsocksR.sh 2>&1 | tee shadowsocksR.log

  如果提示没有wget时使用yum install wget -y即可，一般不会出现。然后会依次提示：

	1.输入ShadowsocksR的密码，即你要设置ssr使用的密码。
	2.选择一个端口：原则上选择1-65535之间皆可。
	3.选择加密方式：输入加密方式前面的数字即可（选择12.chacha20）。
	4.选择协议：和加密方式选择方法相同（选择3.auth_shal_v4）
	5.选择混淆方式（选择2.http_simple）
	6.然后回车即可，等待几分钟后会弹出之前你的密码IP和上述选项，可以选择拍照记下。之后关闭xshell，下载SSR即可。

  然后下载SSR应用即可，官网应该时是不能访问的，好在都是用Github下载的,选择版本下载即可。

  下载之后打开shadowsocksR.exe，然后点击桌面右下角的小飞机,然后点击添加，填入之前的各个信息确认因该就OK啦。

  还有开机启动选项在选项设置中勾选。

  注意基本就是完成了SSR的搭建了。

使用锐速提速

  其实我也不知道有没有用，但是好像确实是有用的，安装之后真的就是看Youtube的时候使用720P不会有卡顿，我也不太清楚是不是那会网快，应该还是有效果的，可以试一下。

	1.更换内核，之前选择的是CentOS7的版本，所以直接打开xshell输入：
	rpm -ivh http://soft.91yun.pw/ISO/Linux/CentOS/kernel/kernel-3.10.0-229.1.2.el7.x86_64.rpm --force
	完成后使用rpm -qa | grep kernel查看是否安装成功。然后重新启动服务器。
	2.重启后输入
	wget --no-check-certificate -O rskernel.sh https://raw.githubusercontent.com/uxh/shadowsocks_bash/master/rskernel.sh && bash rskernel.sh
	期间选项一律按回车即可。等待完成后出现ServerSpeeder is running！字样即可。表示安装完成。

锐速常用命令：

	启动：/serverspeeder/bin/serverSpeeder.sh start
	停止：/serverspeeder/bin/serverSpeeder.sh stop
	状态：service serverSpeeder status
	检查是否有appex0模块：lsmod

netspeeder(便宜好用吧)

  之前那个Vultr搭建再使用锐速确实是速度很快，但是不知道什么原因啊，就是我搭建的那个老是会挂掉(继珂告诉我好像是混淆什么的不太合适，或者是被盯上了我的电脑…难搞了啊)，而且好贵啊，一个月5$，真的是负担不起，所以就再换了这个netspeeder，18.8￥一个月够用了的。而且速度还可以，就是会卡一下，但是整体速度可能是用锐速+Vultr的70%吧。

  NetSpeeder网址，就是直接申请账号，然后直接按照教程安装使用就好了，最好是先充值余额然后再买服务，不然可能会有点问题。

  下载完成后登录即可，然后在那个官网里面直接选择版本，下载然后按照教程就可以弄完了。下面是登录的主界面，也可以在下面的地方换节点。非常的方便(23333333333).
