0.关于民谣记事的一些自己的看法

Topping | 发表于 2019-07-05 | 分类于 记事

我的想法

  我喜欢民谣是在上初三的时候吧，那时候可能就是非常向往那种背着吉他然后去街上流浪(后来还真的去学习了，也没什么结果吧)，心里面装着故乡和心爱的姑娘，也算是一种另类的逃避生活吧，喜欢的姑娘和你没有任何交集，自己的生活又累又没有乐趣，那时候我的小音响里面下了好多的民谣类型(姑且就这样称呼吧)的歌曲，听着宋东野，赵雷，贰佰，陈粒，纣王，李志，马頔，花粥(我那时候真的就开始听了的)等等等等，之后到了高中之后，想听歌也会首先找民谣标题里面的去听，可能是因为听着舒服些。

  可能就是青春期的时候真的会改变一个人性格很多，我记得之前我老是会因为一点小事情就和家里人吵架，但是上高一之后就很少了，一方面是懂了他们的心酸，一方面可能就是青春期太过叛逆，把火泻多了吧。也是从高一开始，真正的喜欢上了民谣，那时候民谣真的很小众了，而且做这种音乐的感觉也真的就是背着吉他流浪，那时候歌曲里面有好多抱怨生活什么的，记得之前宋东野有一首《咬之歌》，还有花粥魔改的《北方汉子》，《啦啦啦德玛西亚》什么的，尧十三的《他妈的》还可以光明正大播放来着，之后就开始在讲故事了吧(就是加入了很多现在看来很民谣的东西吧)，然后民谣就莫名其妙的火了起来，可能是之前好多年的生活积攒下来的故事，能在人群中产生共鸣了？

  他们老是玩笑的讲别让自己喜欢的歌手火起来了，自己还想要偷偷藏着他的歌听呢，但是我并没有这种想法吧，我是喜欢他们的歌曲，会为他们成功感到高兴，也不会在他们落魄的时候去如何抨击他们，我就只是安静的听他们的歌曲，我也觉得这就够了，有歌曲听有时候也挺幸福的嘛！就像宋东野吸毒事件，之前我听他的《安河桥》，《董小姐》…之后听他的《郭源潮》，也就只是知道了这件事情，但是感觉对自己也没有什么多的影响。还有花粥的抄袭事件，前后也就是听歌而已，没什么过多的善恶的想法。其中唯一可惜的算是逼哥的事情吧，可能我消息比较不灵通(平时很少刷微博什么的)，知道事情的时候逼哥的歌已经都下架(三天后？这速度也太快了吧)了，就是也是有点遗憾的其实，因为互联网时代的发展，下载歌曲已经很少见了，最后只是在之前用过的一个小mp4里面找到了《港岛妹妹》和《梵高先生》，用以缅怀那个还债多年的李志吧(之后也会聊逼哥吧，先去努力找些歌)。

  之前也有听过淘漉音乐的音乐故事什么的，觉得很不错，所以也就一直想自己弄一个这种音乐专栏什么的。算是给自己的一个礼物吧，就是能在大学生活中为自己留下一些有意义的记录。说不定以后自己也会从事这个工作呢，哈哈哈哈哈！希望是的吧。
