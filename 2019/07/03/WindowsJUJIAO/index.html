Windows 聚焦

  就是之前有发现新买的笔记本的那个登录时候的图片老是会变，而且都还很好看，就查了下怎么弄下来：

  首先就是要开启Windows聚焦，在个性化的"屏锁界面"选择Windows聚焦的形式来显示屏锁，之后那些图片会存放在C盘的Assets里面，可以网上查一下具体位置进去，进去之后会发现一堆乱码命名的文件，并没有具体的图片。

  所以我们将其复制到另外的地方并使用一个很简单的脚本来使其全部转换为图片格式。代码如下：

cd /d %~dp0
	
ren * *.jpg

  将上述代码保存在FocusCollection.bat文件中，双击运行就可以把上图中的文件转化为具体的图片，但是有一点麻烦的是每次转化完之后.bat文件都会转化为.jpg，需要我们手动改回来。
